# 사업자 진위 조회 및 목록
### 서비스 배경
해당 서비스는 사업장의 휴폐업 정보를 알지 못하여 피해를 보는 사람들이 없도록 하기 위해 만들어졌다.<br>
사업자번호 조회를 통해 휴폐업 사업장을 확인하고, 존재하는 사업장과 존재하지 않는 사업장을 구분하여br>
이들을 한데 모아 목록으로 편리하게 확인하는 것이 가능하다. 해당 프로젝트는 국세청에서 제공하는 오픈API를 활용하였다.
